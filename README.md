## 技术栈
**Vue2**：采用最新Vue2的语法

**Vuex**：实现不同组件之间的状态共享

**vue-router**：单页应用路由管理必备

**axios**：发起http请求

**SASS(SCSS)**：css预处理语言

**Webpack**：自动化构建工具，大部分配置vue-cli脚手架已经弄好了，很方便。

**ES6**：采用ES6语法，这是趋势。 

**localStorage(HTML5)**：本地存储，保存用户个性化设置。😊 

**CSS3**：CSS3动画及样式。

## 使用 Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
